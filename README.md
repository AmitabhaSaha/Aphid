# Aphid

> A lightweight MQTT 3.1.1 client written in pure Swift 3.

## Features:

  - [x] MQTT 3.1
  - [x] MQTT 3.1.1
  - [x] LWT
  - [x] SSL/TLS
  - [ ] Message Persistence
  - [ ] Automatic Reconnect
  - [ ] Offline Buffering
  - [ ] WebSocket Support
  - [x] Standard TCP Support
  - [x] Non-Blocking API
  - [ ] Blocking API
  - [ ] High Availability
